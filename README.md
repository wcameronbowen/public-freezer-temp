# public-freezer-temp

## Use Case
Monitor temperature in freezer (or other appliance) and send text/email alerts when it falls out of range

### Instruction
1. Copy `secrets.py.example` to `secrets.py` and fill out with appropriate credentials/config
2. Copy `secrets.py`, `freezer_bot.py`, and `umail.py` to pico root directory
3. Boot it up!
